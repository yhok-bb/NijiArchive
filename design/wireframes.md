# NijiArchive 画面設計（ワイヤーフレーム）

**作成日:** 2026-07-06
**ステータス:** Draft
**方針:** ペルソナA（みさき）基準で「シンプルで見やすい」を最優先。軸2はデフォルト非表示で視聴体験を邪魔しない。

---

## 1. トップページ（`/`）

```
+--------------------------------------------------------------+
| NijiArchive                            [ログイン] [新規登録]  |
+--------------------------------------------------------------+
| ▼ 新着アーカイブ                                              |
| +----------+  +----------+  +----------+                     |
| | サムネ   |  | サムネ   |  | サムネ   |                     |
| | タイトル |  | タイトル |  | タイトル |                     |
| | 1:30:00  |  | 2:05:00  |  | 0:45:00  |                     |
| +----------+  +----------+  +----------+                     |
|                                                              |
| ▼ 盛り上がったライブ（ハイライト数順）                        |
| +----------+  +----------+  +----------+                     |
| | サムネ 🔥12 |  ...                                          |
| +----------+                                                 |
+--------------------------------------------------------------+
```

- 未ログインでも一覧は見える。クリックでログイン誘導（FR-2-3）
- 「盛り上がったライブ」はハイライト数でソート（Step 3.5以降に表示）

## 2. 視聴ページ（`/videos/:id`）— このサービスの主戦場

```
+--------------------------------------------------------------+
| NijiArchive                                  みさき [ログアウト] |
+--------------------------------------------------------------+
| +------------------------------------+  +------------------+ |
| |                                    |  | コメント          | |
| |          動画プレイヤー             |  | 12:34 最高だった  | |
| |          (hls.js)                  |  | 12:35 ここすき    | |
| |                                    |  | 12:35 🔥🔥🔥      | |
| +------------------------------------+  |       …          | |
| |▶ ──●──────🔥───────🔥──── 1:23/1:30|  +------------------+ |
| |            ↑ハイライトマーカー      |  | [コメントを書く…] | |
| +------------------------------------+  +------------------+ |
| 👁 現在 47人が視聴中                                          |
| タイトル / 説明文                                             |
|                                                              |
| ▼ 関連動画（YouTube）                                        |
| [サムネ|タイトル|チャンネル名 →YouTubeへ] × N               |
|                                                              |
|                                  [🔧 配信を覗く]  ←右下固定  |
+--------------------------------------------------------------+
```

- ハイライトマーカー（🔥）クリックでその位置にジャンプ（US-F-3）
- コメントは再生位置に同期して流れる。投稿は現在の再生位置に紐づく
- 関連動画は埋め込みではなく外部リンク（著作権配慮、ADR-0006）<!-- 要確認: YouTube公式の埋め込みプレイヤー(iframe)は規約上OK。リンクにするか埋め込むかはStep 7で決める -->
- 「配信を覗く」がダッシュボードのトグル（FR-6-5）

## 3. 配信可視化ダッシュボード（視聴ページ内オーバーレイ）

「配信を覗く」クリックでプレイヤー下部に展開。閉じると完全に消える。

```
+--------------------------------------------------------------+
| 🔧 配信の裏側                                        [✕ 閉じる] |
| +---------------------------+  +---------------------------+ |
| | ビットレート推移（あなた）   |  | 全視聴者の画質分布          | |
| |  Mbps                     |  |  1080p ████████ 32人       | |
| |  6 ─╮  ╭──╮               |  |   720p ████ 11人           | |
| |  3   ╰──╯  ╰────          |  |   360p █ 4人               | |
| |    (リアルタイム更新)       |  |  (5秒ごと更新)             | |
| +---------------------------+  +---------------------------+ |
| | ABR切り替えログ                                            | |
| | 12:34:56 720p → 1080p (6.0Mbps) ・ CDN: HIT               | |
| | 12:33:10 1080p → 720p (3.0Mbps) ・ CDN: HIT               | |
| |   (直近20件)                                               | |
| +------------------------------------------------------------+ |
| ℹ この仕組みの解説 → [Zenn記事] [GitHub]                       |
+--------------------------------------------------------------+
```

- グラフは Chart.js <!-- 要確認: 候補。Stimulus + 軽量ライブラリなら uPlot も。Step 4.5 で決定 -->
- 「この仕組みの解説」リンクがペルソナB→記事・リポジトリへの導線（UC-2-5）

## 4. ログイン / 同時ログイン制限ブロック画面

```
ログイン(/session/new):            ブロック時(視聴ページ):
+----------------------+          +------------------------------+
| email    [        ]  |          | ⚠ 他のデバイスで視聴中です    |
| password [        ]  |          | このアカウントは既にPCで視聴  |
| [ログイン]           |          | 中のため再生できません。      |
| 新規登録はこちら      |          | (視聴中の端末を閉じると解除)  |
+----------------------+          +------------------------------+
```

- ブロックは「後から来た方」に出す。先行セッションは継続（UC-3）

## 5. 管理画面（`/admin/videos`）

```
+--------------------------------------------------------------+
| 管理: 動画一覧                            [+ 新規アップロード] |
| +----------------------------------------------------------+ |
| | タイトル       | 状態        | 操作                       | |
| | 3rdライブDay1  | published   | [編集] [非公開に] [削除]   | |
| | 3rdライブDay2  | processing ⏳| (変換中 2/3解像度)         | |
| | 4thライブDay1  | failed ⚠    | [再変換] [削除]            | |
| +----------------------------------------------------------+ |
+--------------------------------------------------------------+
```

- アップロードは direct upload + 進捗バー。変換状態（videos.status）が一覧で見える（US-A-2）
